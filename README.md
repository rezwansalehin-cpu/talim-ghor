# Talim Ghor Website

Static public profile website for Talim Ghor in Hatibandha, Sakhipur, Tangail.

## Files

- `index.html`: page structure, copy, public links, and source list.
- `styles.css`: responsive visual design.
- `script.js`: mobile navigation, active section state, reveal effects, and image fallback behavior.
- `assets/images/`: local copies of approved images from the official KAMPS Free Medical Camp page.
- `talim-ghor-website-intel.md`: research notes and claim-safety boundaries.

## Preview Locally

From this folder:

```bash
python3 -m http.server 8793
```

Then open:

```text
http://127.0.0.1:8793/
```

## Publishing Notes

- GitHub Pages can publish this site directly from the repository root.
- Keep `.nojekyll` in the repository so GitHub Pages serves the static assets as-is.
- Use the official spelling `Talim Ghor` in public-facing copy.
- Keep Green Haven Agriculture as a stewardship section, not the main identity of the site.
- Do not add event registration, fee, exact timing, regular testing, dialysis, or permanent clinic claims unless confirmed separately.
- Keep the public source list visible so visitors can verify the story.
- If replacing the gallery, prefer owned family/venue/scenic images and keep source credits for any public archive images.
